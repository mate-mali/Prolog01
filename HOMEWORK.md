# Zrobic dsterownika ladowania gdzie
- poziom naladowania bardzo niski, niski, sredni, wysoki bardzo wysoki 
- temperatura sterownika bardzo niski, niski, sredni, wysoki bardzo wysoki 
- temperatura baterii bardzo niski, niski, sredni, wysoki bardzo wysoki 
- pobierany prąd / prąd rozładowania bardzo niski, niski, sredni, wysoki bardzo wysoki 
- napięcie na panelach bardzo niski, niski, sredni, wysoki bardzo wysoki 

# Wartoscia wynikowa ma byc dobrany prad ladowania na baterię 
zrobic zestaw pietnastu + ruli uzywajac mamdani któe:
- jeśli temperatura baterii lub sterownika jest wysoka, to maly prad ladowania, 
- jesli napiecie na panelach bardzo wysokie to duzo laduj
- jesli prado rozladowania wysoki to wtedy wtedy wiekszy prad ladowania 
- jesi poziom naladowania bardzo wysoki to mniejszy prad ladowania (bo lifepo4 i podobny szajs)
